Приложение "Заметки"

Структура:
Код создает оконное приложение с двумя колонками: слева текстовое поле, справа - список заметок, кнопки управления заметками, список тегов, поле для ввода тега и кнопки управления тегами.
Данные: Заметки и их теги хранятся в словаре data, который сохраняется и загружается из файла notes.json.
Функциональность:
Создание, удаление и сохранение заметок.
Просмотр текста заметок и их тегов.
Добавление и удаление тегов у заметок.
Поиск заметок по тегам.
Интерфейс: Приложение написано с использованием модуля PyQt6  и  json на языке Python.
Используются виджеты QTextEdit (для текста заметки), QListWidget (списки заметок и тегов), QPushButton (кнопки), QLineEdit (поле для ввода тега) и QLabel (метки).
Этот код представляет собой довольно полную систему заметок, с графическим интерфейсом и базовыми функциями для работы с записями и их тегами.
